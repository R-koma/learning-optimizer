# API設計: エンドポイント一覧

## 概要

FastAPIの自動生成ドキュメント（`/docs`）で確認することが前提だが、PoCの時よりも複雑になったため、API設計を行い、整理する。

---

## REST API

### ノート

| メソッド | パス             | 説明                                                    | 認証 |
| -------- | ---------------- | ------------------------------------------------------- | ---- |
| GET      | `/api/notes`     | ユーザーのノート一覧を取得                              | 必須 |
| GET      | `/api/notes/:id` | ノート詳細を取得                                        | 必須 |
| PATCH    | `/api/notes/:id` | ノートを更新（復習後の内容更新等）                      | 必須 |
| DELETE   | `/api/notes/:id` | ノートを削除（関連するfeedback, review_scheduleも削除） | 必須 |

### 復習スケジュール

| メソッド | パス                        | 説明                                                                       | 認証 |
| -------- | --------------------------- | -------------------------------------------------------------------------- | ---- |
| GET      | `/api/review-schedules`     | 今日の復習タスク一覧を取得（`status=pending AND next_review_at <= now()`） | 必須 |
| PATCH    | `/api/review-schedules/:id` | 復習完了マーク + 次回復習日の更新                                          | 必須 |

### フィードバック

| メソッド | パス                       | 説明                                         | 認証 |
| -------- | -------------------------- | -------------------------------------------- | ---- |
| GET      | `/api/notes/:id/feedbacks` | ノートのフィードバック履歴を取得（時系列順） | 必須 |

---

## WebSocket

| パス        | 説明                                    | 認証                      |
| ----------- | --------------------------------------- | ------------------------- |
| `/ws/text`  | テキストチャット（学習対話 / 復習対話） | ハンドシェイク時にJWT検証 |
| `/ws/audio` | 音声チャット（STT → 対話 → TTS）        | ハンドシェイク時にJWT検証 |

### WebSocketメッセージ型（概要）

**クライアント → サーバー:**

| type             | 説明                        |
| ---------------- | --------------------------- |
| `start_learning` | 新規学習開始（topicを送信） |
| `start_review`   | 復習開始（note_idを送信）   |
| `user_message`   | ユーザーの発話              |
| `end_session`    | セッション終了              |

**サーバー → クライアント:**

| type                 | 説明                            |
| -------------------- | ------------------------------- |
| `assistant_message`  | LLMの応答                       |
| `note_generated`     | ノート生成完了（note_idを返却） |
| `feedback_generated` | フィードバック生成完了          |
| `session_ended`      | セッション終了確認              |
| `error`              | エラー通知                      |
